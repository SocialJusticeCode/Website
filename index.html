ALL NON-CIS PEOPLE HAVE THE RIGHT TO USE GITHUB AND PARTICIPATE IN CODING IN AN ENVIRONMENT WHERE THEY WILL NOT BE HURT.

SocialJusticeCode is an organization that attempts to help the oppressed in the coding community by calling out hate given by cis people, and help oppressed individuals have a safe experience using GitHub and have a good time coding.
